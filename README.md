## Code for: Generalization Bounds for Robust Contrastive Learning: From Theory to Practice [(arxiv)](https://arxiv.org/pdf/2311.09671)
- Some of the source code is derived from [ngoctnq/ssa-clr](https://github.com/ngoctnq/ssa-clr).
- SAM code adapted from [davda54/sam](https://github.com/davda54/sam).
-Data folder is set at args/constrastive.py --data_folder
- contrastive.py: SSL training
- linear_correct_bn.py: Linear evaluation
- contrastive_ckpt_d.py: training D from a given ckpt.
- To run the code, please follow the commands in scripts/run.sh
